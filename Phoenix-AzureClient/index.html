Phoenix-Azure

A proof of concept application demonstrating healthcare data integration and FHIR interoperability.

This application showcases two key capabilities:

  1. Connecting to the AmazingCharts API endpoint (https://apiserviceswin20250318.azurewebsites.net/api) to import patient data directly
  2. Transforming that same data into standardized FHIR resources for healthcare interoperability

The local API server acts as a proxy to handle CORS issues while connecting to the Azure-hosted endpoint.

Patient List (AmazingCharts Direct)

View patient data in a traditional healthcare interface format. This view connects to the AmazingCharts API endpoint (https://apiserviceswin20250318.azurewebsites.net/api) and displays patient information in a user-friendly tabular format.

This demonstrates standard API integration for healthcare applications.
